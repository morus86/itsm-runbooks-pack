# Incident Runbook

## Goal
Restore service quickly and safely.

## Triage checklist
- What is impacted? (service, users, scope)
- When did it start?
- Any recent change?
- Check monitoring/alerts/logs

## Communication
- Update ticket with facts (no guesses)
- If needed: notify stakeholders

## Resolution
- Apply mitigation
- Validate recovery
- Monitor for regression

## Post-incident
- Capture timeline
- Decide if a Problem record is needed
